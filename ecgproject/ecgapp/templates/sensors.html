Sensors & Equipment

AD8232 ECG Sensor

The AD8232 is a specialized analog front-end sensor used to capture ECG (electrocardiogram) signals. It provides:

  • High signal quality with low noise
  • Compact and low-power design
  • Perfect for wearable or portable heart monitoring
  • Captures electrical activity of the heart through electrodes

ARDUINO-UNO Microcontroller

The Arduino Uno is a popular microcontroller board based on the ATmega328P. In this project:

  • It receives analog ECG data from the AD8232 sensor module
  • Processes and digitizes the signal using its onboard ADC (Analog-to-Digital Converter)
  • Sends the data to a computer or server via serial communication (USB) for further processing

ECG Electrodes

ECG electrodes are placed on the skin to detect the heart’s electrical signals. These are:

  • Disposable wet gel electrodes for better conductivity
  • Connected to AD8232 via color-coded leads
  • Essential for capturing clean signals from the body

Software Architecture

  • Django Backend
    • Real‑time ECG graph visualization using Python scripts
    • Interactive dashboard built with Django templates and CSS styling
  • Deep Learning Models
    • CNN and BiLSTM architectures for anomaly detection with Attention Modules and Focal Loss
    • Designed to identify arrhythmias and other ECG irregularities
